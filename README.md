## Getting Started

```bash
# Install vue-cli and scaffold boilerplate
$ npm install -g @vue/cli
$ npm install -g @vue/cli-init

$ vue init kdydesign/vue-electron my-project
$ cd my-project
$ npm install
$ npm run dev
```
